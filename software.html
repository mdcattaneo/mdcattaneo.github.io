Regression Discontinuity Designs

  1. Package rdrobust: RD Plots, estimation, and inference.
  2. Package rdhte: Estimation and inference for heterogeneous treatment effects.
  3. Package rdlocrand: Local randomization methods.
  4. Package rddensity: Manipulation testing.
  5. Package rdpower: Power and sample size calculations.
  6. Package rd2d: Estimation and inference for boundary discontinuity designs.
  7. Package rdmulti: Estimation and inference with multiple cutoffs and multiple scores.

Treatment Effects, Causal Inference, and Attention Models

  1. Package binsreg: Visualization, estimation and inference using binscatter methods.
  2. Package scpi: Estimation and inference for synthetic control methods.
  3. Package mvte (coming soon): Estimation and inference for multi-valued treatment effects under ignorability.
  4. Package ramchoice: Nonparametric estimation and inference in random limited attention settings.

Nonparametrics and Semiparametrics

  1. Package nprobust: Kernel density and local polynomial regression methods.
  2. Package lpdensity: Local polynomial distribution/density regression methods.
  3. Package lpcde: Local polynomial conditional distribution/density regression methods.
  4. Package lspartition: Partitioning-based least squares regression methods.
